# geojson-to-wkt

PWA made with Vue 3, Vite, Pinia and Leaflet to convert between GeoJSON and WKT formats.

![Preview](https://i.imgur.com/OuTbA5Z.png)

## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm dev
```

### Compiles and minifies for production
```
pnpm build
```
